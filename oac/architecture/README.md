# Архитектурные решения NovaMedika2

**Назначение папки:** Документы с описанием архитектурных решений для проекта NovaMedika2 в контексте соответствия требованиям ОАЦ.

---

## 📋 Назначение

Здесь хранятся ADR (Architecture Decision Records) и описания ключевых технических решений, влияющих на соответствие требованиям ОАЦ класса 3-ин.

**Важно:** Это НЕ нормативные документы (те в [`../docs/`](../docs/)), а **проектные технические решения**.
