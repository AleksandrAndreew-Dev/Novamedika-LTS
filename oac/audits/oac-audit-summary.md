### 2. Шифрование БД (0% готово)
**Проблема:** Персональные данные хранятся в открытом виде

**Решение:** Использовать pgcrypto для PostgreSQL
```sql

-- Шифрование чувствительных полей
UPDATE users SET phone_encrypted = pgp_sym_encrypt(phone, 'key');
```

### 5. Резервное копирование (10% готово)
**Проблема:** Нет автоматических backup БД и конфигураций

**Решение:** Настроить cron job для backup
```bash
#!/bin/bash
# /usr/local/bin/backup.sh
DATE=$(date +%Y%m%d_%H%M%S)
docker exec postgres-prod pg_dump -U $POSTGRES_USER $POSTGRES_DB | \
  gzip > /backups/db_$DATE.sql.gz
  
# Хранить последние 365 дней
find /backups -name "db_*.sql.gz" -mtime +365 -delete
```
