## 📁 Куда сохранять документы?

### 1️⃣ Нашел новый закон/приказ в интернете?
```
📍 Сохранить в: origin-docs/
✅ Формат: .md (текст) или .pdf (резервная копия)
📝 Пример: origin-docs/new-law.md
```

### 2️⃣ Создал документ для compliance проекта?
```
📍 Сохранить в: oac/docs/
✅ Формат: XX-название.md (с нумерацией!)
📝 Пример: oac/docs/14-new-policy.md
⚠️ Важно: Обновить CHANGELOG.md
```

### 3️⃣ Провел аудит или тестирование?
```
📍 Сохранить в: oac/audits/
✅ Формат: YYYY-MM-DD_тип_описание.md
📝 Пример: oac/audits/2026-04-21_vuln-scan-q2.md
```

### 4️⃣ Написал инструкцию для персонала?
```
📍 Сохранить в: oac/guides/
✅ Формат: topic-guide.md
📝 Пример: oac/guides/incident-response-guide.md
```

### 5️⃣ Составил план работ?
```
📍 Сохранить в: oac/planning/
✅ Формат: plan-name.md
📝 Пример: oac/planning/compliance-roadmap.md
```

## ⚡ Быстрые команды

### Проверить наличие всех нормативных документов:
```bash
py check_normative_docs.py
```

### Посмотреть статус Git:
```bash
git status
```

### Добавить и закоммитить изменения:
```bash
git add -A
git commit -m "docs: краткое описание изменений"
```
