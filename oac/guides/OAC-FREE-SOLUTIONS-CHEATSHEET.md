## 🛠️ Рекомендуемый стек бесплатных инструментов

### 1. Централизованное логирование 🔴
**ELK Stack (Elasticsearch + Logstash + Kibana)**
```bash
# Запуск одной командой
docker-compose -f docker-compose.logging.yml up -d
```
- ✅ Хранение логов ≥ 1 год
- ✅ Полнотекстовый поиск
- ✅ Визуализация и алерты
- ✅ Ресурсы: ~2-3 GB RAM

**Альтернатива (легче):** Loki + Grafana (~1 GB RAM)

---

### 2. Шифрование БД 🔴
**pgcrypto (встроен в PostgreSQL)**
```sql
-- Установка (1 команда)
CREATE EXTENSION pgcrypto;

-- Шифрование поля
UPDATE users SET phone_encrypted = pgp_sym_encrypt(phone, 'key');
```
- ✅ Соответствует требованиям криптографии ОАЦ
- ✅ Шифрование на уровне полей
- ✅ $0 стоимость

---

### 3. Резервное копирование 🔴
**Bash скрипт + cron**
```bash
# Ежедневный backup в 2:00
0 2 * * * /usr/local/bin/backup.sh
```
- ✅ Автоматическая ротация (хранить 365 дней)
- ✅ Компрессия (gzip)
- ✅ Локально или rsync на другой сервер
- ✅ Можно добавить бесплатный cloud (Yandex.Disk 10GB)

---

### 4. Антивирус 🟡
**ClamAV**
```bash
apt install clamav
# Сканирование каждый час
0 * * * * clamscan -r /var/lib/docker/volumes
```
- ✅ Регулярные обновления баз
- ✅ Интеграция с приложением
- ✅ Ресурсы: ~512 MB RAM

---

### 5. IDS/IPS 🟡
**Fail2Ban**
```bash
apt install fail2ban
# Защита от brute-force SSH, HTTP
```
- ✅ Блокировка IP после неудачных попыток
- ✅ Легковесный (~50 MB RAM)
- ✅ Простая настройка

**Продвинутый уровень:** Suricata (полноценный IDS/IPS)

## ⚡ Быстрый старт (1 неделя)

### День 1-2: Основа
```bash
# 1. Установить антивирус
apt install clamav && systemctl enable clamav-freshclam

# 2. Настроить защиту от brute-force
apt install fail2ban && systemctl enable fail2ban

# 3. Создать backup скрипт
cat > /usr/local/bin/backup.sh << 'EOF'
#!/bin/bash
DATE=$(date +%Y%m%d)
docker exec postgres-prod pg_dump -U $POSTGRES_USER $POSTGRES_DB | \
  gzip > /backups/db_$DATE.sql.gz
find /backups -name "db_*.sql.gz" -mtime +365 -delete
EOF
chmod +x /usr/local/bin/backup.sh

# 4. Настроить cron
echo "0 2 * * * /usr/local/bin/backup.sh" | crontab -
```

### День 3-5: Логирование
```bash
# Развернуть ELK Stack
git clone <repo>
cd novamedika
docker-compose -f docker-compose.logging.yml up -d

# Настроить Filebeat/Promtail для сбора логов
```

### День 6-7: Мониторинг и шифрование
```bash
# 1. Развернуть Prometheus + Grafana
docker-compose -f docker-compose.monitoring.yml up -d

# 2. Включить шифрование в PostgreSQL
docker exec postgres-prod psql -U $POSTGRES_USER -c "CREATE EXTENSION pgcrypto;"

# 3. Запустить первое сканирование OWASP ZAP
docker run -t owasp/zap2docker-stable zap-baseline.py \
  -t https://api.spravka.novamedika.com
```

## ⚠️ Когда нужны платные решения

### Ситуации, где free НЕ подходит:

1. **Требуется сертификация ФСТЭК/ОАЦ**
   - Некоторые средства защиты должны быть сертифицированы
   - Проверить актуальный перечень в Приказе №66

2. **Нет экспертизы в команде**
   - Open-source требует времени на изучение
   - Если нет DevOps инженера → рассмотреть managed решения

3. **Критический SLA 99.99%**
   - Платные решения включают техподдержку 24/7
   - Для стартапа/MVP free достаточно

4. **Большой масштаб (>100k пользователей)**
   - При росте нагрузки может потребоваться enterprise поддержка
