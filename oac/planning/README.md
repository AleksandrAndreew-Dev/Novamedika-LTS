## 📝 Как использовать эти документы

### Для менеджера проекта:
1. Распечатать [oac-compliance-checklist.md](oac-compliance-checklist.md)
2. Назначить ответственных за каждую задачу
3. Обновлять статус еженедельно
4. Использовать [OAC-VISUAL-PROGRESS.md](OAC-VISUAL-PROGRESS.md) для отчетов

### Для команды разработки:
1. Изучить [OAC-AUDIT-SUMMARY-FOR-TEAM.md](OAC-AUDIT-SUMMARY-FOR-TEAM.md)
2. Взять задачи из чек-листа согласно своей роли
3. Следовать инструкциям из [guides/](../guides/)
4. Отмечать выполнение в чек-листе

### Для руководства:
1. Просматривать [OAC-VISUAL-PROGRESS.md](OAC-VISUAL-PROGRESS.md) еженедельно
2. Контролировать бюджет по таблице выше
3. Участвовать в kickoff meeting (документ для команды)
4. Принимать решения на основе gap analysis
