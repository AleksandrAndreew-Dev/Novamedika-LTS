# Requirements - Требования ОАЦ

Эта папка содержит документы с требованиями Оператора Активных Компонентов (ОАЦ) и планы по их внедрению.

---

## 📄 Документы

### 1. [oac-requirements.md](oac-requirements.md)
**Полные требования ОАЦ к системе NovaMedika2**

Содержит:
- Детальный анализ всех требований класса 3-ин
- Технические требования к защите информации
- Организационные требования
- Требования к документации
- Критерии соответствия

**Использование:** Основной документ для понимания, что требуется от системы.

---

### 2. [oac-sprint.md](oac-sprint.md)
**План спринтов по внедрению требований**

Содержит:
- Разбивку работ на спринты
- Приоритизацию задач
- Оценку времени и ресурсов
- Зависимости между задачами

**Использование:** Для планирования работ по compliance.
