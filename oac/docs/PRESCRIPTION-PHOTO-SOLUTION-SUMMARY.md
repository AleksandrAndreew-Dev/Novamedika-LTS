## ✅ РЕШЕНИЕ

### Гибридная архитектура

```
Telegram Bot (только текст + кнопки)
    ↓
Кнопка "Загрузить рецепт"
    ↓
Web Application на сервере РБ
    ↓
Авторизация по SMS → Согласие на специальные ПД → Загрузка фото
    ↓
Хранение на серверах РБ (beCloud/hoster.by) с AES-256
    ↓
Фармацевт видит только в режиме просмотра (без скачивания)
    ↓
Автоудаление через 48 часов
```
