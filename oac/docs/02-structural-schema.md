## 4. Физические границы информационной системы

**Границы ИС:**

- **Внешняя граница:** Входной трафик через внешний IP-адрес сервера (порты 80, 443)
- **Внутренняя граница:** Docker сеть `traefik-public` (bridge)
- **Физическая граница:** Сервер [ДАТА-ЦЕНТР]
