## 2. ЧТО ТАКОЕ COOKIE-ФАЙЛЫ

### 2.1. Определение

Cookie-файлы — это небольшие текстовые файлы, которые сохраняются на вашем устройстве (компьютере, смартфоне, планшете) при посещении web-сайта. Они позволяют сайту запоминать ваши действия и предпочтения (такие как логин, язык, размер шрифта и другие настройки отображения) на определенный период времени, чтобы вам не приходилось повторно вводить их при каждом возвращении на сайт или переходе между страницами.

### 2.2. Аналогичные технологии

Помимо cookie-файлов, мы используем аналогичные технологии:
- **LocalStorage** — локальное хранилище браузера для сохранения настроек
- **SessionStorage** — временное хранилище для текущей сессии
- **IndexedDB** — база данных в браузере для офлайн-функциональности
- **Service Workers** — фоновые скрипты для кэширования ресурсов

### 2.3. Классификация cookie-файлов

| Тип | Описание | Примеры использования |
|-----|----------|----------------------|
| **По сроку действия** | | |
| Сессионные | Удаляются при закрытии браузера | Идентификатор сессии |
| Постоянные | Хранятся до истечения срока или удаления пользователем | Настройки языка, тема оформления |
| **По источнику** | | |
| Собственные (first-party) | Устанавливаются нашим сайтом | localStorage с настройками |
| Сторонние (third-party) | Устанавливаются сторонними сервисами | ❌ Не используем |
| **По назначению** | | |
| Строго необходимые | Требуются для работы сайта | Авторизация, безопасность |
| Функциональные | Запоминают ваш выбор | Язык, регион |
| Аналитические | Помогают понять поведение пользователей | ❌ Не используем без согласия |
| Маркетинговые | Используются для рекламы | ❌ Не используем |

## 3. КАКИЕ COOKIE-ФАЙЛЫ МЫ ИСПОЛЬЗУЕМ

### 3.1. Строго необходимые cookie-файлы

Эти cookie-файлы необходимы для функционирования сайта и не требуют вашего согласия.

| Название | Тип | Срок хранения | Цель |
|----------|-----|---------------|------|
| `session_id` | Сессионный | До закрытия браузера | Идентификация сессии пользователя |
| `auth_token` | Постоянный | 7 дней | JWT токен авторизации (для фармацевтов) |
| `csrf_token` | Сессионный | До закрытия браузера | Защита от CSRF-атак |
| `cookiesAccepted` | Постоянный | 1 год | Запись о принятии Политики cookie |

**Правовое основание:** Статья 6 Закона №99-З — исполнение договора, законные интересы оператора (безопасность).

---

### 3.2. Функциональные cookie-файлы

Эти cookie-файлы запоминают ваш выбор и улучшают функциональность сайта. Требуется ваше согласие.

| Название | Тип | Срок хранения | Цель |
|----------|-----|---------------|------|
| `language` | Постоянный | 1 год | Предпочитаемый язык интерфейса (ru/by/en) |
| `theme` | Постоянный | 1 год | Тема оформления (light/dark) |
| `consent_preferences` | Постоянный | 1 год | Ваши настройки согласий на обработку ПД |
| `last_search_query` | Сессионный | До закрытия браузера | Последний поисковый запрос (для удобства) |

**Правовое основание:** Статья 6 Закона №99-З — согласие субъекта персональных данных.

---

### 3.3. Аналитические cookie-файлы

**❌ Мы НЕ используем аналитические cookie-файлы по умолчанию.**

Если в будущем потребуется внедрить аналитику (например, Яндекс.Метрика, Google Analytics), это будет сделано только после получения вашего явного согласия.

---

### 3.4. Маркетинговые cookie-файлы

**❌ Мы НЕ используем маркетинговые cookie-файлы.**

Мы не занимаемся таргетированной рекламой и не передаем данные рекламным сетям.

## 4. АНАЛОГИЧНЫЕ ТЕХНОЛОГИИ ХРАНЕНИЯ ДАННЫХ

### 4.1. LocalStorage

Мы используем LocalStorage для сохранения следующих данных:

```javascript
// Примеры данных в LocalStorage
{
  "cookiesAccepted": "true",           // Принятие Политики cookie
  "consents": {                         // Согласия на обработку ПД
    "privacyPolicy": true,
    "dataProcessing": true,
    "securityProtection": true,
    "transboundaryTransfer": true       // Для Telegram-бота
  },
  "userPreferences": {                  // Настройки пользователя
    "language": "ru",
    "theme": "light"
  }
}
```

**Срок хранения:** До удаления пользователем или 1 год с последнего посещения.

**Правовое основание:** Согласие субъекта ПД (статья 6 Закона №99-З).

---

### 4.2. SessionStorage

Используется для временного хранения данных текущей сессии:
- Промежуточные результаты поиска
- Временные фильтры
- Данные форм до отправки

**Срок хранения:** До закрытия вкладки браузера.

---

### 4.3. Service Workers и Cache API

Используются для:
- Кэширования статических ресурсов (CSS, JS, изображения)
- Обеспечения работы в офлайн-режиме
- Ускорения загрузки страниц

**Данные:** Только технические файлы, не содержащие персональные данные.

**Правовое основание:** Законные интересы оператора (улучшение производительности).

Вы можете управлять cookie-файлами через настройки вашего браузера:

| Браузер | Инструкция |

### Приложение A. Технические детали cookie-файлов

#### A.1. Структура cookie-файлов

```javascript
// Пример cookie-файла
{
  "name": "cookiesAccepted",
  "value": "true",
  "domain": ".novamedika.com",
  "path": "/",
  "expires": "2027-05-20T00:00:00Z",
  "secure": true,          // Только HTTPS
  "httpOnly": false,       // Доступен через JavaScript
  "sameSite": "Strict"     // Защита от CSRF
}
```

#### A.2. Безопасность cookie-файлов

Все cookie-файлы защищены:
- **Secure flag** — передаются только по HTTPS
- **SameSite attribute** — защита от CSRF-атак
- **Шифрование** — чувствительные данные шифруются перед сохранением
