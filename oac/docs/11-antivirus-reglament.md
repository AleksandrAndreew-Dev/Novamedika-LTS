## 1. ОБЩИЕ ПОЛОЖЕНИЯ

## 2. СРЕДСТВА ЗАЩИТЫ

### 2.1. ClamAV — Антивирус

| Параметр | Значение |
|----------|----------|
| Контейнер | `clamav-prod` |
| Образ | `clamav/clamav:1.3` |
| Назначение | Сканирование uploaded CSV файлов |
| Расписание сканирования | Перед каждой обработкой CSV (вызов из Celery) |
| Обновление сигнатур | Автоматическое, каждые 4 часа |
| Действие при обнаружении | Удаление файла + логирование |

### 2.2. Fail2ban — Защита от брутфорса

| Параметр | Значение |
|----------|----------|
| Контейнер | `fail2ban-prod` |
| Образ | `crazymax/fail2ban:latest` |
| Назначение | Блокировка IP при множественных неудачных попытках |
| SSH: maxretry | 3 попытки, бан на 2 часа |
| HTTP auth: maxretry | 5 попыток, бан на 1 час |
| Bad bots: maxretry | 3 попытки, бан на 24 часа |

## 3. ПОРЯДОК СКАНИРОВАНИЯ

### 3.1. Сканирование CSV файлов

Перед каждой обработкой CSV в Celery:

```python
# tasks/tasks_increment.py
import subprocess

def scan_csv_with_clamav(file_path):
    result = subprocess.run(
        ["bash", "agent/oac/clamav-scan.sh", file_path],
        capture_output=True, text=True, timeout=60
    )
    if result.returncode == 2:
        raise ValueError(f"CSV file infected: {file_path}")
    return True
```

### 3.2. Обновление сигнатур

- Автоматическое обновление ClamAV (freshclam) каждые 4 часа
- Проверка: `docker exec clamav-prod freshclam --dry-run`
