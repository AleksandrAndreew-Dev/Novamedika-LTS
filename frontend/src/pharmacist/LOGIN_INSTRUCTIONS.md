# 📱 Инструкция по входу в панель фармацевта

## 🔐 Как войти в панель управления

### Шаг 1: Откройте бота в Telegram
1. Найдите бота **NovaMedika** в Telegram
2. Откройте диалог с ботом

### Шаг 2: Нажмите кнопку панели
В меню бота нажмите кнопку:
```
💼 Панель фармацевта
```

### Шаг 3: Введите Telegram ID
Откроется окно входа. Вам нужно ввести ваш **Telegram ID**.

**Как узнать свой Telegram ID:**
1. Найдите бота [@userinfobot](https://t.me/userinfobot) в Telegram
2. Напишите ему любое сообщение (например, `/start`)
3. Бот ответит с вашим ID (число, например: `123456789`)
4. Скопируйте это число

### Шаг 4: Войдите
1. Вставьте ваш Telegram ID в поле ввода
2. Нажмите кнопку **"Войти"**
3. Готово! Вы вошли в панель управления 🎉

## ❓ Часто задаваемые вопросы

### Почему нужно вводить ID каждый раз?
Для безопасности. Ваш Telegram ID подтверждает вашу личность.

### Что делать, если вход не работает?
1. Проверьте, что вы правильно ввели Telegram ID (только цифры)
2. Убедитесь, что вы зарегистрированы как фармацевт в системе
3. Если проблема сохраняется - обратитесь к администратору

### Можно ли войти с другого устройства?
Да! Просто повторите шаги выше на любом устройстве.

### Сколько длится сессия?
- **Access token**: 30 минут
- **Refresh token**: 7 дней
- После истечения access token система автоматически обновит его
- Через 7 дней нужно будет войти заново

### Что означает статус "Онлайн"?
Когда вы онлайн, вы получаете уведомления о новых вопросах от пользователей в реальном времени.
