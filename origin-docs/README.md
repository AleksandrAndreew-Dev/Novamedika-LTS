# Оригинальные нормативные документы (origin-docs)

**Назначение папки:** Хранение **оригинальных текстов** законов, указов, приказов и разъяснений регуляторов РБ в формате `.md` или `.pdf`. **Без нашей аналитики** — только первоисточники.

---

## 📋 Правила

- ✅ Только **оригинальные тексты** нормативных актов
- ✅ Формат: **.md** (предпочтительно) или **.pdf** (резервная копия)
- ❌ НЕ размещать аналитику, интерпретации или проектные документы (для этого — `oac/`)
- ✅ Каждый документ должен иметь чёткое название с номером и датой
